1、安装git  
2、 创建一个git 仓库 
	git init  

3、 把文件添加到暂存区
	git  add 文件名称 

4、  将版本库里面的文件添加到仓库 
	git commit -m “ 提交修改的说明”
5、查看当前状态 
	git status 
6、 插看修改内容 
	git diff --cached   文件名
7. 查看提交的版本
	git  log 
8、选择版本 
	1）、git reset  --hard head^ 退回上一个版本 想退回几个版本就加几个^
	2)、 git reset  --hard  commit前7位编码
9、 查看操作记录
	git reflog
10、 撤销工作区未提交之前的内容 
	git checkout -- 文件名称
