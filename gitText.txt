1、安装git  
2、 创建一个git 仓库 
	git init  

3、 把文件添加到暂存区
	git  add 文件名称 

4、  将版本库里面的文件添加到仓库 
	git commit -m “ 提交修改的说明”
5、查看当前状态 
	git status 
6、 插看修改内容 
	git diff --cached   文件名
7. 查看提交的版本
	git  log 
8、选择版本 
	1）、git reset  --hard head^ 退回上一个版本 想退回几个版本就加几个^
	2)、 git reset  --hard  commit前7位编码
9、 查看操作记录
	git reflog
10、 撤销工作区未提交之前的内容 
	git checkout -- 文件名称 
11、 删除文件 
	git rm 文件名 
12.链接远程仓库
	git remote add origin https://github.com/myyangyue/gitStudy.git
13.把本地仓库推送到远程仓库上去
	git push -u origin master
14、展示一个本地项目 
	1）、设置里面，找到GitHub Pages 将Source 设置成主分支
	2）、打开 github 账号和厂库名称：https://github.com/myyangyue/myStudyGit/（如何有文件夹 name就是项目名称，默认打开页面是index.html)
15、创建子分支 
	git branch  子分支名称
16、 将head指向子分支 
	git  checkout  分支
	git switch  分支
17、   创建子分支 并且指向子分支 
	git checkout -b 子分支
	git switch -c 子分支
18、 查看当前分支 
	git branch 
19、 合并分支
	git merge  子分支名称
20、删除子分支 
	git branch -d 子分支名称
